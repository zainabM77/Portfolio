General-Purpose Processor Design

The lab aims to design a General-Purpose Processor with an Arithmetic and Logic Unit (ALU) using VHDL and implement it on an FPGA board. The processor consists of four main components: Procuring input data, Storage Unit (Register), Control Unit, and ALU Core. Two 8-bit register units are created to store inputs A and B. On the rising edge of the clock signal, the register reads input values and outputs them on the next rising edge. Input data A and B are obtained using the last four digits of the student ID, for this lab the value for A 00110010 is and B is 01011001. VHDL code is written for the latches, and symbols are generated for the final circuit.

BankAccount Application

The banking system project is designed to facilitate efficient management of financial transactions for customers and managers. It features a use case diagram that outlines interactions, including customer login, money deposits, withdrawals, and online purchases, as well as managerial tasks. The class diagram highlights key classes such as BankAccount, Customer, and Manager, with the BankAccount class serving as a superclass for various account tiers (Silver, Gold, Platinum) that implement different privileges. The project also employs the State design pattern to manage account behaviors based on membership levels, ensuring a secure and user-friendly experience through a JavaFX interface.

Amplifier Design

The aim of this project is to design, simulate, and test a single-supply, multistage, inverting transistor amplifier meeting specific requirements. Amplifiers are integral to electronic systems, providing signal amplification for various applications. The project entails selecting appropriate configurations, calculating component values, and simulating the circuit to ensure compliance with specifications. Through theoretical analysis, simulation using MultiSim, and experimental testing, the goal is to develop an amplifier design that demonstrates proficiency in analog circuit design principles.
